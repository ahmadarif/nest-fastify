## Description

NestJS + Fastify + Fastify File Upload Adapter

## Installation

```bash
$ npm install
```

## Prepare the environment
```bash
$ cp .env.example
```

## Running the app

```bash
# development
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Build the app

```bash
$ npm run build
```
