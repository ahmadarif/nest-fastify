## Description

NestJS with Fastify + Distributed Socket.io

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run serve

# production mode
$ npm run start:prod
```

## Build the app

```bash
$ npm run build
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
